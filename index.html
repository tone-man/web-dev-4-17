Regular Expression Practice

This Regular Expression checker checks for the following.

  1. All the words of any length that start and end with the letter d.

  2. All of the variations of the word "letter".

  3. One or more digits followed by the letter "w" and then 0 or more digits.

  4. Any number of X's, followed by exactly 3 O's.

  5. A phrase that is four words separated by single spaces, each word five or fewer letters.
